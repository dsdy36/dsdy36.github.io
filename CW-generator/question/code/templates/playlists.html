Playlist Management

Use the forms below to manage your playlists. Each operation is performed independently.

Create New Playlist

Enter a name to create a new empty playlist.

Rename Playlist

Select a playlist and enter a new name to rename it.

Delete Playlist

Select a playlist to delete it permanently. This action cannot be undone.

Remove Tracks by Genre

Remove all tracks of a specific genre from a selected playlist.

Add Tracks by Genre

Add all Tracks of a specific genre to a selected playlist. Tracks that are already in the playlist will be ignored.
